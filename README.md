# team5-java-cli-board
- 계좌생성 : add
입력 : 계좌번호, 초기금액, 비밀번호, 이름, 계좌코드

- 입금: push
입력 : 계좌, 비밀번호, 이름, 입금금액

- 출금 : pull
입력 : 계좌, 비밀번호, 이름, 출금금액

- 이체 : trans
입력 : 출금할 계좌, 비밀번호, 금액, 이름, 입금할 계좌

- 계좌조회 : list
입력 : 계좌번호, 비밀번호
출력 : 이름, 금액, 입금계좌, 개설날짜, 
      입금내역, 입금액, 입금날짜, 입금자, 출금날짜, 출금액 출금자, 출금계좌, 계좌코드(출금시 입금내용 *****, 입금시 출금내용*****)

- 계좌폐기 : remove
입력 : 계좌, 비밀번호 

- 언어 : java
